void LoginPage::loginUser ( )
slot

Gets called after a user has attempted to login. using the json.cpp utility class:

  • loads the users.json file using json.getJsonDocument()
  • checks if the attempted login was successful using json.checkUser(usersArray,username,password) if successful, the curUser instance gets updated to the currently logged in user else, curUser becomes NULL
